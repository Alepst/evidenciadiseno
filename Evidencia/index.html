Evidencia Etapa 4

Durante esta etapa aprendimos a cómo hacer un sitio web con un menú horizontal. A lo largo de esta también aprendimos algunas funciones de JavaScript, como las funciones predefinidas, pues estas convierten valores de un tipo a otro. También vimos cómo hacer uso de las funciones de manipulación de cadenas, funciones de manipulación de arrays, funciones de matemáticas; después nos enseñaron cómo hacer funciones por nuestra cuenta, lo cual es relativamente fácil para nosotros ya que llevamos otras materias que también requieren declarar funciones.

En este semestre vimos muchas cosas sobre JavaScript, un lenguaje de suma importancia debido a que es uno de los lenguajes más usados en esta época moderna. Aprendimos el uso de las alertas emergentes, como lo pueden ser el alert(), confirm() y prompt(). Aprendimos también cómo escribir código HTML dentro de un archivo Javascript, sobre los tipos de variables como Number, String, Boolean, Undefinided, Null y Symbol. Aprendimos también cómo declarar objetos en JavaScript,, sobre algunas palabras clave para declarar variables con distintas funciones y sobre los arreglos.

En la etapa 2 aprendimos sobre los distintos tipos de operadores, también sobre las estructuras condicionales y estructuras cíclicas, como For y While. También vimos el uso de getElementById, el cual sirve para devolver una referencia a un elemento por su ID.

En la etapa 3 vimos sobre algunos eventos de JavaScript, como onclic, onFocus, Reset, onmousedown y onmouseup, etc. También vimos las utilidades básicas para los formularios, validación de formularios y cómo hacer un formulario.

Podemos concluir que ya sabemos las bases de JavaScript como lenguaje de programación utilizado para el desarrollo y diseño de páginas y sitios web.
